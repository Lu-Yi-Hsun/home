5G 三大場景：eMBB、mMTC 和 URLLC

eMBB 對應的是 3D / 超高清影像等大流量行動寬頻業務

mMTC 對應的是大規模物聯網業務

URLLC 對應的是如無人駕駛、工業自動化等需要低時延高可靠連接的業務。
