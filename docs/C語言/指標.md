## function pointer

### 例題一

(\*(void(\*)())0)()

![](/images/pointer.png)

!!! note "用途"
    指向其他的函數 其他寫法會比較漂亮

??? note "(*(void(*)())dd_prt)()把function pointer 指向其他的函數"
    ```c   
        #include <stdio.h>
        void dd(int a){
            printf("%d",a);
        }
        int main()
        {
        int a=3;
        void (*dd_prt)(int)=&dd;//宣告function pointer 
        (*(void(*)())dd_prt)(5566);
                     //^^^^這裡可以寫記憶體位置(所以該位置指向dd function所以我們能操作dd)      
            return 0;
        }


    ```
